A importância do Box Model

Para projetar um layout CSS, seja para uma página web simples ou um sistema web completo é fundamental que o desenvolvedor entenda com detalhes como os elementos HTML são apresentados no viewport do usuário.

Todo elemento HTML é uma "caixa retangular" cujos atributos básicos são: margens, bordas, espaçamento interno e o conteúdo propriamente dito. Esses atributos são estilizáveis e essas caixas podem ser empilhadas até formarem a página completa. 🇧🇷
