O QUE SÃO UNIDADES DIVERSIFICADAS?

As unidades diversificadas são disciplinas obrigatórias da matriz curricular dos cursos técnicos integrados ao ensino médio. Cada estudante deve fazer pelo menos uma unidade diversificada por semestre. Elas possuem temas organizados conforme proposta de cada professor, sendo o estudante quem deve escolher, entre as que estão sendo ofertadas no semestre, qual ele quer cursar.

REGRAS:

Para fazer a inscrição, cada estudante deve: Clicar no botão "Conhecer" a cima, ler os resumos das Unidades Diversificadas para conhecê-las; Observar o horário que as Unidades Diversificadas serão ofertadas; Escolher uma das Unidades Diversificadas para cursar no semestre 2023/1; Clicar no botão "Inscrever-se" da diversificada desejada e entrar no formulário para realizar a matrícula;

Caro o estudante, ao clicar no nome da diversificada e visualizar a mensagem "Vagas esgotadas, escolha outra unidade", isso significa que as vagas disponibilizadas para esta diversificada já foram preenchidas, cabendo ao estudante escolher outra unidade diversificada para cursar no semestre 2023/1.
